Python Crash Course

Eric Matthes

Read

A solid intro to Python that doesn’t waste time. It balances explanation and hands-on practice well, and the projects feel like real challenges at that stage of learning. The syntax and core concepts are covered clearly, without overcomplicating. Would’ve liked a bit more depth on testing and structuring larger codebases, but for its scope, it delivers.

Python Programming: An Introduction to Computer Science

John Zelle

Read

Slower paced, but really solid if you're after fundamentals. It leans more into CS concepts than just Python syntax, which makes it great for understanding how things work under the surface. Not the fastest way to build something, but it gives you the right mental models early on.

Django 5 By Example

Antonio Melé

Read

Very practical, straight into building real stuff. The examples are solid and scale up nicely, so you get a sense of how to structure bigger Django projects. Not much hand-holding, but if you already know Python, it's a great way to learn by doing.

Python for Data Analysis

Wes McKinney

Currently Reading

I’ve only made it through the first bit, but it’s already super practical. It doesn’t waste time on basics, it just jumps straight into pandas and real data work. Feels dense at times, not a fast read.

Effective pandas Patterns
for Data Manipulation

Matt Harrison

Currently Reading

Read about half of it, and it’s been really useful so far. It doesn’t teach pandas from the ground up, but focuses on doing things more efficiently and cleanly.

UNIX: A History and a Memoir

Brian Kernighan

Read

It’s more historical than technical, but still a great read if you’re into how things came to be. Kernighan keeps it personal and clear, walking through UNIX’s development without getting too dense. Not something you study from, but definitely worth reading to understand the roots of modern computing.

Data Structures and Algorithms in Python

Michael T. Goodrich | Roberto Tamassia | Michael H. GoldWasser

Reading

It provides a solid introduction to fundamental data structures and algorithm analysis, with a strong emphasis on asymptotic complexity and object-oriented design in Python. The explanations are formal, and the implementations are clean and consistent. So far, it’s a solid reference for serious study.
